# satomatashikitask

統合型プロジェクト・タスク管理システム

複数のプロジェクト、開発タスク、SNSマーケティング、KPI目標を一元管理するダッシュボードアプリケーション

## 主な機能

### プロジェクト管理
- 複数プロジェクトの一覧表示・進捗管理
- プロジェクトステータス（進行中・完了・保留）
- タグ・カラー管理
- マイルストーン・タスク管理

### タスク管理（Kanban方式）
- ボード・リスト・カード管理
- ドラッグ&ドロップによる並び替え・移動
- カレンダービュー（期限管理）
- カードの詳細カスタマイズ（色・装飾）
- タグ機能

### KPI・目標追跡
- 数値目標の設定（SNS投稿、売上、PVなど）
- 実績記録・履歴管理
- 進捗率の可視化
- 複数周期対応（毎週・毎月・毎年など）

### 外部連携
- **YouTube統計**: チャンネル統計・グラフ分析
- **Discord通知**: Webhook連携、スケジュール通知

### その他
- ドキュメント管理（プロジェクト・ボード・カード単位）
- SNS投稿計画管理

## セットアップ

```sh
npm install
npm run dev
```

http://localhost:5173

### その他のコマンド

```sh
npm run db:check   # データベース状態確認
npm run db:reset   # データベースリセット
```

## 技術スタック

- **フロントエンド**: SvelteKit (Svelte 5)
- **スタイリング**: Tailwind CSS 4
- **データベース**: Cloudflare D1 (SQLite)
- **UI**: Lucide Icons, Chart.js
- **認証**: bcryptjs
- **その他**: svelte-dnd-action (ドラッグ&ドロップ)

## ユースケース

- YouTuber / クリエイターの複合業務管理
- マルチプロジェクトのコンテンツ企画
- SNS・YouTube・ブログの統合管理
- 開発タスクとマーケティング活動の一元化
