# satomatashikitask

個人用のTrello風タスク管理ツール

## 技術スタック

- **フロントエンド**: SvelteKit
- **スタイリング**: Tailwind CSS
- **データベース**: Cloudflare D1 (SQLite)
- **ストレージ**: Cloudflare R2
- **デプロイ**: Cloudflare Pages

## 開発環境のセットアップ

```sh
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```
